GLab
Eduational Hardware Simulation Framework

Established in 2022, the GLab is a research group at the University of Southern California that focuses on operating system and computer architecture research. The group is led by Dr. Andrew Goodney and is part of the USC Computer Science Department.

Our Focus

NetBSD Kernel Development

Customized NetBSD kernel for educational purposes. The kernel is designed to be simple and easy to understand, making it a great tool for teaching operating system concepts.

M68k Adaptation

M68k microprocessor simulation with C/C++ framework. Designed to be portable and modularized for easy integration.

RISC-V Adaptation

RISC-V microprocessor simulation using a C/C++ framework. The RISC-V framework is designed to work and experiment with the latest technology, in addition to the existing M68k framework.
