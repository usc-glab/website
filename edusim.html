EDUSim

A framework for kernel and hardware simulation for educational purposes.

Project Overview

The framework is developed to provide a platform for hardware and kernel simulation. It is designed to be modular and extensible, allowing for easy integration of new hardware and kernel modules. Right now it is divided into two main parts: the kernel and the hardware. The framework now provides support RISC-V and M68K architectures.
Device: essentially an abstraction of a hardware component. It can be a simple device like a timer or a more complex devices.
System Board: a collection of devices that are connected to a bus. The system board is the main component of the hardware simulation.
Mushashi: a simple m68k emulator that is used to run the kernel (luna68k currently).
